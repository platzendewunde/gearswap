📥 How to Download and Setup

  1. Save this HTML page to your computer (Right-click → Save As → "download_files.html")
  2. Create project folder: Create a new folder called wrestling-results-processor
  3. Copy each file: For each file below, copy the content and save it with the exact filename shown
  4. Create folder structure: Create the subfolders as shown in the structure below
  5. Run setup: Follow the setup instructions in the README
